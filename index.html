Primeiramente você precisa saber qual trilha você quer seguir, existem várias (ainda não adicionadas ao site, em breve serão), como por exemplo o desenvolvimento Web, onde você pode ser tanto Front-end, Back-end ou Full-stack

Talvez você queira partir logo para o Back-end, aprendendo linguagens como C#, C++, Python ou Java (cursos para essas linguagens serão adicionados no futuro)

Mas nesse início minha recomendação é que você se volte para o desenvolvimento Web, aprendendo o core de JavaScript e posteriormente o React Native para aplicações Mobile

Bom, se você é um completo iniciante é bem provável que nada do que eu falei até agora fez qualquer sentido para você, mas enfim, a primeira trilha é a Full-stack Web.
